INSTRUZIONI PER RUNNARE IL PROGETTO DATAVIS 

- Scaricare e installare un ambiente di lavoro ad esempio Pycharm 

- Aprire il il progetto 

- Aprire il Terminal 

- Creare un ambiente virtuale con questo commando : python -m venv +Nome dell ambiente virtuale 

- Attivare l ambiente virtuale con il commando : .\nome dell ambiente virtuale\Scripts\activate

- Installare  le librerie di questo progetto con commando : pip install -r .\requirements.txt

- Per eseguire il progetto , spostarsi su tesi e eseguire il commando :  python.exe .\manage.py runserver.

NB: Una volta terminato l utlimo commando cioe python.exe .\manage.py runserver , cliccare su link generato e aggiungere al  percorso della pagina che si apre l estensione seguente /dashboard/api/dash
